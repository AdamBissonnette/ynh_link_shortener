Ceci est un modèle générique pour empaqueter des applications Node.js pour YunoHost.

Il fournit un échafaudage complet avec :
- Configuration du manifeste YunoHost
- Scripts d'installation, de mise à niveau, de sauvegarde et de restauration
- Service systemd avec durcissement de sécurité
- Reverse proxy Nginx avec support WebSocket
- Application Express.js exemple

Parfait pour déployer rapidement n'importe quelle application Node.js sur YunoHost.
